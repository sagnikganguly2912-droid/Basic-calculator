# Basic-calculator

Project Title

Basic Calculator

Introduction

The Basic Calculator is a web-based application developed to perform fundamental arithmetic operations efficiently. It provides a simple and user-friendly interface for quick mathematical calculations.

Problem Statement

Manual calculations can be time-consuming and prone to human error. A digital calculator helps ensure accuracy and saves time during basic mathematical operations.

Objectives

To perform basic arithmetic operations

To display results instantly

To provide a clean and intuitive user interface

To handle invalid inputs safely

Technologies Used

HTML

CSS

JavaScript

Functional Description

Users interact with the calculator through on-screen buttons. JavaScript captures user inputs and processes calculations based on selected operators. The result is displayed immediately.
